High-level page map for LancerScape (blockchain freelancing marketplace with milestone escrow and AI-assisted dispute resolution).

## Client

- **Dashboard**
  - Short description: Overview of active projects, milestones, balances and recent activity.
  - Key actions: View project statuses and deadlines; release or dispute milestone payments; message freelancers and review proposals.

- **Project Marketplace / Browse Freelancers**
  - Short description: Find freelancers or post projects with requirements and milestones.
  - Key actions: Search/filter by skill/rating/price; save/shortlist candidates; open project details or start a posting flow.

- **Create Project (New Job Wizard)**
  - Short description: Step-by-step form to define scope, milestones, payment terms, and escrow settings.
  - Key actions: Define milestones and deliverables; set escrow terms; publish project and invite proposals.

- **Project Detail (Client view)**
  - Short description: Single project page showing milestones, proposals, contract, chat and escrow status.
  - Key actions: Review proposals and award a freelancer; approve/contest deliverables; view escrow balance and history.

- **Payments & Escrow**
  - Short description: Manage escrow deposits, refunds, and payment history for projects.
  - Key actions: Deposit funds; release funds or request refunds; view transaction/fee breakdown.

- **Messaging / Inbox**
  - Short description: Communication hub for project conversations and notifications.
  - Key actions: Send/receive messages and attachments; link messages to milestones/disputes; archive/flag threads.

- **Profile & Billing**
  - Short description: Client account, payment methods, billing receipts, and preferences.
  - Key actions: Update billing methods; view receipts; configure notifications.

## Freelancer

- **Dashboard**
  - Short description: Snapshot of active contracts, pending milestones, earnings, and tasks.
  - Key actions: See upcoming deliverables; accept/decline projects; track payout status.

- **Job Marketplace / Discover Projects**
  - Short description: Browse available projects posted by clients.
  - Key actions: Submit proposals; save projects; view client profiles and ratings.

- **Proposal Composer**
  - Short description: Create and submit proposals with milestones, pricing and timelines.
  - Key actions: Draft milestone schedule and cost; attach portfolio items; submit and track status.

- **Project Detail (Freelancer view)**
  - Short description: Contract view with milestone requirements, files, chat, and escrow info.
  - Key actions: Upload deliverables; request milestone review/payment release; open a dispute if needed.

- **Submit Work & Deliverables**
  - Short description: Upload/submit work tied to a milestone for client approval.
  - Key actions: Attach files/notes; mark submitted and request review; track feedback.

- **Earnings & Payouts**
  - Short description: Overview of cleared funds, pending escrow releases, payout methods.
  - Key actions: Configure payout method; withdraw balance; view payout history.

- **Profile & Portfolio**
  - Short description: Public freelancer profile, skills, ratings, and portfolio items.
  - Key actions: Update skills and rates; upload portfolio samples; manage visibility.

## Dispute System

- **Dispute Center (entry)**
  - Short description: Unified place to start and track disputes tied to projects/milestones.
  - Key actions: Open dispute linked to milestone/payment; view status; access timeline and required actions.

- **Open Dispute (wizard)**
  - Short description: Guided flow for submitting a dispute with reason, affected milestone, and evidence.
  - Key actions: Select category; upload evidence; submit for AI mediation or human arbitration.

- **Dispute Detail**
  - Short description: Timeline view of the dispute, participants, evidence, and current state.
  - Key actions: Add evidence/comments; acknowledge outcomes; escalate to arbitration.

- **AI Mediation Console**
  - Short description: AI-assisted decision helper that summarizes evidence and proposes resolutions.
  - Key actions: Run automated analysis; view suggested resolutions and confidence; initiate settlement or forward to human review.

- **Arbitration Panel / Human Review**
  - Short description: Human adjudication interface for escalated disputes.
  - Key actions: Inspect case and evidence; record ruling; trigger escrow actions (release/refund).

- **Dispute History & Analytics**
  - Short description: Archive of past disputes and analytics for patterns and trust metrics.
  - Key actions: Filter/search historical disputes; export reports; view system metrics.

## Admin

- **Admin Dashboard**
  - Short description: Operational overview for platform health, funds in escrow, and active disputes.
  - Key actions: Monitor escrow balances; see alerts and KPIs.

- **User & Role Management**
  - Short description: Manage client and freelancer accounts, roles, bans and appeals.
  - Key actions: Suspend/reactivate accounts; inspect user history; approve/revoke privileges.

- **Escrow & Finance Management**
  - Short description: Tools to audit escrow flows, reconcile transactions, and manage fees.
  - Key actions: Inspect/reconcile escrow accounts; adjust balances with audit trail; configure fee schedules.

- **Dispute Oversight & Arbitration Tools**
  - Short description: Admin interface to oversee escalated disputes and human arbitration.
  - Key actions: Assign arbitrators; audit AI outputs; generate reports.

- **Policy & Content Moderation**
  - Short description: Configure policies, remove prohibited content, and manage reports.
  - Key actions: Update terms and enforcement workflows; review flagged content.

- **Audit Logs & Compliance**
  - Short description: Immutable logs and exportable trails for compliance and investigations.
  - Key actions: Search/export audit trails; trigger investigations.

---

## Shared (common across roles)

- **Authentication (Login / Signup / Wallet Connect)**
  - Short description: Entry points for signing up and authenticating (email, SSO, crypto wallet).
  - Key actions: Register/verify identity; connect wallet; enable 2FA.

- **Notifications Center**
  - Short description: Central list of in-app, email, and webhook notifications.
  - Key actions: View/filter/dismiss notifications; configure channels.

- **Activity Feed / Timeline**
  - Short description: Global or per-project timeline for recent events.
  - Key actions: Drill into events; subscribe/unsubscribe to feeds.

- **Wallet & Account Security**
  - Short description: Wallet management and security settings for on-platform crypto operations.
  - Key actions: Connect wallets; manage recovery guidance; configure withdrawal limits.

- **Help / Support / Documentation**
  - Short description: Knowledge base, FAQ, contact support and developer docs.
  - Key actions: Search docs; open support tickets; access API/webhook docs.

- **Transaction History & Receipts**
  - Short description: Unified ledger of payments, escrow deposits/releases, fees and payouts.
  - Key actions: Filter/export history; download receipts.

- **Settings & Preferences**
  - Short description: Account-level preferences shared by all user types.
  - Key actions: Update personal info; configure appearance and integrations.
