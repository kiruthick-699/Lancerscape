# Lancerscape

A blockchain-based freelancing marketplace with milestone escrow and AI-assisted dispute resolution.

## Project Structure

```
Lancerscape/
├── frontend/          # Next.js 14 frontend with Tailwind & shadcn/ui
├── backend/           # Node.js/Express backend with AI services
└── contracts/         # Solidity smart contracts (Hardhat)
```

## Technologies

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS, shadcn/ui
- **Backend**: Node.js, Express, AI services
- **Blockchain**: Solidity, Hardhat, Ethereum

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn
- Git

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Backend Setup
```bash
cd backend
npm install
npm run dev
```

### Smart Contracts Setup
```bash
cd contracts
npm install
npx hardhat compile
```

## Features

- Blockchain-based escrow for milestone payments
- AI-assisted dispute resolution
- Smart contract integration
- Real-time notifications
- Secure file uploads
